    # 关于2018-2019学年综合测评评分标准(细则）

    # 思想品德分：

    # 思想政治观念（15分）

    • 党员15
    • 预备14
    • 积极分子13（包括上学年党课结业优秀证获得者）
    • 其余团员11
    • 群众9

    TIP

    纪律观念、集体观念、基础文明修养：根据对应情况得分

    # 纪律观念（14分）

    • 优：严格要求自己，自觉遵纪守法，坚决同各种违法违纪行为作斗争，得分为12～14分。
    • 良：能遵纪守法，未发现有违法违纪行为，得分为8～11分。
    • 中：基本上能遵纪守法，偶有违纪行为，但能及时改正，得分为5～7分。
    • 差：法纪观念淡薄、纪律性差，对自己要求不严、纠正违纪现象不及时、不彻底，得分0～2分。

    # 集体观念（13分）

    • 优：主动关心集体，积极参加集体活动，为集体活动出谋献策；爱护集体荣誉，热心为集体服务，积极参与各专项活动及其后勤工作，不计较个人得失得分为11～13分。
    • 良：能关心集体，积极参加集体活动，爱护集体荣誉，能为集体服务。得分为8～10分。
    • 中：基本上能参加集体活动，但为集体服务不积极、不主动，得分为5～7分。
    • 差：集体观念淡薄，不关心集体活动、不愿为集体服务，得分为0～2分。

    # 基础文明修养（13分）

    • 优：举止文明，明礼诚信、尊敬师长、团结同学，善于与人相处，人际关系融洽，敢于批评不文明行为，得分为11～13分。
    • 良：讲文明，懂礼貌，爱护公物，节约水电，对他人损坏公物和浪费水电等不良行为敢于批评，得分为8～10分。
    • 中：一般能爱护公物，偶有损坏公物和浪费水电行为能及时改正，得分为5～7分。
    • 差：缺乏修养，不讲文明，弄虚作假，损坏公物，如随地吐痰，常在课桌上、墙壁上乱涂乱画，向墙壁上踢球，常开长流水、长明灯等，得分为0～2分。

    # 宿舍制度（15分）

    • 上一学年获得“校文明宿舍”称号的四个宿舍（15分）
    • 上一学年宿舍卫生检查中每月获得“文明宿舍”（有收过奖品）的宿舍（13分）
    • 其余宿舍（10分）

    TIP

    注：生活部提供名单

    # 社会实践（15分）

    • 优：参加校或院三下乡活动获得先进个人，积极分子表彰，或者获得优秀社会调查报告表彰，得分13～15分。（实践部提供名单）
    • 良：参加校或院三下乡活动，或者实习，或者上交社会调查报告两篇以上的，得分9～12分。（三下乡和调查报告实践部提供名单，实习要实习单位开证明截图）
    • 中：有上交社会调查报告的，得分4～8分。（实践部提供名单）
    • 差：社会实践能力差。不按时参加社会实践或虽参加社会实践，态度不端正、作风散漫、纪律性较差，得分为0～4分。

    # 奖分（最高15）

    1. 校级六大学生组织（团务中心、学生会、艺术团、社团联合会、红十字、青协）的主席（书记）加8分。

    2. 校级六大学生组织（团务中心、学生会、艺术团、社团联合会、红十字、青协）的副主席（副书记）、院系级学生组织的主席（主任）加7分。

    3. 校级六大学生组织（团务中心、学生会、艺术团、社团联合会、红十字、青协）的部长、院系级学生组织的副主席（副主任）加5分。 校级六大学生组织的副部长、院系级学生组织的正部长，助班、班长、团支书、学习委员加4分。

    4. 校级六大学生组织（团务中心、学生会、艺术团、社团联合会、红十字、青协）的干事、院系级学生组织的副部长（班委）加3分。 院系级学生组织的干事（信息员）加2分。

    5. 无任何职位不加分. （身兼多职者，按最高职务计分，校级的要出示工作牌证明，院级的学生会，分团委，青协提供名单）

    6. 党团知识竞赛：一、二、三等奖，分别加思想分2，1.5，1（参赛者多加0.5分），以及优秀组织奖、最佳风采团支部、最具魅力团支部、最佳人气奖（各0.5），

    7. 团务日志：一，二，三分别加思想分2，1.5，1，代表院参赛的班级加多1分。

    8. 团课的奖项各加0.5分思想分

    9. 团日活动:一、二、三等奖，分别加思想分1.5，1，0.5，其他奖项加文体分0.5分

    10. 五四评优：个人获奖者加2分思想分，班级获奖全班加1分思想分。

    11. 院报投稿0.2/篇、发表0.5/篇

    12. 军训征文投稿0.2/篇、（一等奖2分、二等奖1.5分、三等奖1分）

    13. 新生杯征文投稿0.3/篇、（一等奖1.5分、2等奖1分，3等奖0.5分）

    14. 白云简报投稿0.2分/篇，发表1分

    15. 参加暑期校或院三下乡活动1分，优秀调查报告（获校级一等奖思想加1.5分，二等奖加1.2，三等奖加1，优秀奖加0.8）暑期社会实践优秀个人或寒假科技入户先进个人（校级的思想加1分）

    16. 献血4分，广东福彩公益奖2分，（献全血4分，成分血6分）

    17. 校文明宿舍（宿舍成员每人）1分，上一学年宿舍卫生检查中每月获得“文明宿舍”的宿舍（宿舍成员每人）0.5分，广交会志愿者（需有文件证明，加1分），新生迎新志愿者（加1分）

    18. 获市、学校、学院先进个人（校级如优秀团干、市级如广州志愿者）分别加（市级5分，校级2分，院级一分）

    19. 校广播台09—10年度优秀节目主持人（需有宣传部盖章文件证明，加1分）校广播台“优秀策划”（需有宣传部盖章文件证明，加1分）

    20. 学校各社团“工作积极分子”、“组织工作积极分子”（需有团委盖章文件证明，加1分）

    21. 参加志愿者活动（加1分）优秀青年志愿者（加1分）清明参加黄花岗烈士扫墓（加1分）

    22. “西关小屋”志愿者团队，正式注册成为广州志愿者（加1分） 奖分一律要有证明，担任学校干事的或学院干事要有工作证或者其他证明，获奖的要有证书或者或将其他证明，证明全部打包成word文档，每个人一个文件夹标注班级姓名。

    # 学业分

    # 学习成绩

    重要！

    不清楚的记得问！截一学年的绩点表

    学习成绩=（必修课和专业选修课学分绩点之和÷相应学分之和+5）×9 以上学分绩点不包括公共选修课。

    公共选修课成绩如果达到良好或 80 分，每科按 0.5 分加分。 补考重修的一科扣2分，依次叠加（没有下限）

    # 奖励分（最高分10分）

    1. 公共选修课成绩如果达到良好或80分，每科按0.5分加分，上限1分

    2. 上一学年参加英语四级统考成绩达425分及以上者，加3分；学生参加英语六级统考成绩达425分及以上者加4分。学生通过计算机一级考试者加2分，通过二级者，加3分。**（出示成绩单截图、证书照片）

    提示

    在教务处的 查看等级考试 一栏可以看到自己的等级成绩。

    1. 通过专业相关等级考试按等级（初级或初级以上）依次加1-2分（须有证书证明）；

    2. 通过辅修考试（一门加0.8分）

    3. 创新基金正式结题的成员（项目主持人加2分，两名组员各加1分）或（所有成员各加0.5分），结题以结题报告上交到分团委负责的部门为准。以下四条，参赛者可获相应参赛资格加分，

      • 院系级加0.1分，
      • 校级加0.2分，
      • 省级加0.3分，
      • 国家级加0.4分（如果重复参加，不累加，按最高等级加分），

      注意

      获奖也是按最高等级加分

    4. 参加数学建模

      • 初赛获奖

        • 一等奖加3分
        • 二等奖加2分
        • 三等奖加1分
        • 优秀奖加1分

      • 省级数学建模大赛获奖

        • 一等奖加5分
        • 二等奖加4分
        • 三等奖加3分

      注意

      加分不累计，按最高奖计分

    5. 各种专业知识竞赛（例如ACM竞赛，统计调查大赛，全国英语口语竞赛、）、 实践技能竞赛、“挑战杯”学生课外学术科技作品竞赛及创业设计竞赛、大学生职业规划大赛的正式参赛者（不含工作人员）， 院（系）级竞赛加0.1分，校级加0.2分，省级加0.3分，国家级加0.4分；获一、二、三等奖者，院（系）级竞赛分别加1.5、1、0.5分，校级分别加3、2、1分，省级分别加5、4、3分，国家级分别加7、6、5分。优秀奖（仅指不分等级的比赛）按各级别的二等奖计。（出示获奖证明）

    6. 在各级公开发行的刊物（有CN或ISSN登记号）上发表学术论文者，核心刊物第一作者计6分，一般刊物计4分，第二作者减半加分。学生在科研、教学实习或其他活动中取得重大成果或写出高水平论文，经有关专家鉴定或有关部门证明，可得3分，水平特别高者，可酌情给更高分。参加课外技能培训和学习获得各类职业资格证书者，可计1～2分。学生获得国家发明专利或实用新型专利者，加6分。

      注意

      1.院报的文章，编辑部提供名单

      2.学院以外的文章，出示相关证明

    7. 辅修一门课程成绩合格者，加0.8分。 加分超过10分，按10分计。

      注意

      1、如果有获奖的话，不加参赛分。

      2.学院以外的文章，出示相关证明

    8. 某个学院主办，其他学院可参与的比赛按校级加分，例如统计调查大赛，ACM大赛、数学建模等。 扣分按照汇编手册分数扣分，

    # 文体表现分

    # 体育课分 身体健康监测达标成绩*60%；

    # 经常性文体活动分（10分）（自评）

    • 优：有三条或者以上的加分，得分 9～10分。
    • 良：有两条加分，得分8分。
    • 中：有一条加分，得分 4～7分。
    • 差：没有加分的， 0～3分。

    # 奖励分

    1. 积极参加院、校及省以上文体演出、比赛、礼仪和主持活动者，每项分别加院级0.5、校级1、省级2分；

    2. 积极参加院（系）级文体比赛（院级辩论赛、朗诵大赛、 K赛、宿舍文化节、趣味数学竞赛、班服设计大赛、司仪礼仪大赛、秋季运动会）获奖者或获奖班级，每项第一名加3分，第二名加2分，第三名加1分（以后名次按0.2分递减。最佳辩手/优秀个人加1分，可累加）

    3. 参加校级文体比赛（如团日活动）获奖者，每项第一名加6分，第二名加5分，第三名加4分，第四名3分，第五名2分，第六名1分，第七名加0.7分，第八名加0.4分。

      提示

      同类比赛中，校级比赛获奖者按校级文体比赛加分，不累加院级文体比赛分

    4. 参加校组织海报评比设计大赛（加1分）

    5. 参加校级演讲比赛及获奖（参加加1分，获奖加2分）

    6. 参加全国大学生英语知识能力竞赛（参加者加2分。第一名加8分，第二名加6分，第三名加4分，不累加）

    7. 英语口语比赛获奖（按参加“演讲比赛”项目标准加分，获“优秀奖”等非名次奖项的按三等奖标准加分）

    8. 团务日志（按参加校级文体比赛获奖者标准加分）

    9. 校心理健康咨询中心举办的心理剧大赛（加1分）

    10. 三下乡表彰大会获“优秀调研奖”， “最佳爱心奖”， “优秀创意奖”的加1分；

    11. “幸福广东-幸福仲园”获奖（第一名加3分，第二名加2分，第三名一份，以此类推减0.2分，不累加）

      注意

      所有非专业相关的竞赛类都划分为文体分，专业相关竞赛则归为学业分。

    12. “仲恺”、“香凝”杯获奖（按综合测评规定加分）

    13. 校体育节比赛获奖（按综合测评规定加分）

    14. 学校“仲恺杯”,“香凝杯”各类球赛“工作积极分子”（加3分,需有团委提供盖章文件）

    15. 校、院运动会参加通讯员、播音员、彩旗队、方阵、啦啦队（校级加2分/院系加1分）

    16. 校、院运动会领队、裁判（校2分/学院1分）

    17. 校、院系乒乓球及其他球类比赛获奖（按综合测评规定加分）

    加分超过30分，按30分计。

    # 文体表现的扣分

    无故不参加集体文体活动者每次扣5分。 体育成绩不及格（不达标）扣2分。

    # 注意事项（必看）

    1. 综合测评总分计算：思想分20%，学业分70%，文体分*10%

    2. 参与综合测评各项均为上一学年的，不要混淆了！

    3. 该项综合测评每年均要进行，关系到存档、评奖，务必如实填写！

    4. 各项证明资料要按思想分、学业分、文体分各种归类整理注明，整理为三个文档（各项证明可找活动主办方、负责人开证明或者教务网截图）

    如下所示：

    1. 评分参照以上标准，如有参加以上之外的活动或比赛获奖者，需给予相关证明可另外加相应学分。

    2. 所有加分项均需出示相关证明，未出示则不可加分。
